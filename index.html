  • CISA Known Exploited Vulnerabilities Catalog
  • Rapid 7 Metasploit Modules
  • Project Discovery Nuclei Templates
  • EPSS

The list is updated hourly and contains over 5,000 CVEs. The output is available in this CSV file. The projects code is available in this GitHub Repo.

About
Khulnasoft.com is an early-stage open-source project by KhulnaSoft Lab to help collect, combine, and distribute high-quality open-source vulnerability intelligence to help organizations decide how to prioritize their limited patching cycles.
